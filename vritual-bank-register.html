Wallet registration code explained

What is a wallet registration code?

A wallet registration code is a twenty nine digit unique code created for you after the creation of your wallet. It allows you to sign up for a bank account without reusing your email or username.

Do I need wallet registration code?

The short answer is no. You only need a wallet registration code if you created a wallet first and are then signing up for a virtual bank.

If you are registering for a bank account first, you can simply continue using your username, email address, and password, no registration code is required. If you have already signed up but choose not to use your registration code, you can still register using a different email address and username. Just keep in mind that this may mean having more details to remember later.

When is a registration code needed?

The registration code is only needed if you created a wallet before registering for a bank account. When you have created a wallet, your email address and username are saved and linked to you. These details can only be registered once. The registration code allows the system to recognise that you have already created a wallet.

Why can’t I just use the same email and username again?

Reusing registered details without an extra check would make it easier for someone else to attempt to claim an account using information that doesn’t belong to them. The registration code avoids this by confirming your existing wallet setup without exposing any sensitive information.

What happens when I enter the registration code?

The code is checked instantly, without refreshing the page. Once confirmed, your email and username fields are locked, and you can safely set your password to complete your wallet registration.
